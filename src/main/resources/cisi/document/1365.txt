  A file management system can provide a powerful search tool for a libraryreference group.  The system described produces both current awareness andretrospective searches from several diverse data bases.  The same querylanguage can be used to interrogate all data bases in either on-line or batch mode. Searches can be made for any word, word root, phrase, or number in anypart of any entry.  Citations selected by coordination of terms can be eitherprinted or used to custom-make new machine-readable files.  Costs arereasonable; an average batch mode search requires 11 seconds of CDC 6600 time.
